#Signal Generator using MATLAB
## Description:
This project generates a customizable continuous-time signal composed of multiple segments.
Each segment can have a different signal type (DC, ramp, sinusoidal, polynomial, or exponential), defined over user-specified time intervals.
## Features:
-Generate signals with multiple breakpoints
-Support for different signal types:
    -DC
    -Ramp
    -Sinusoidal
    -Polynomial
    -Exponential
-Apply signal operations:
    -Amplitude scaling
    -Time reversal
    -Time shifting
    -Time expansion
    -Time compression
Automatic plotting of the signal before and after operations
## How It Works:
1.Enter sampling frequency and time range
2.Define breakpoints (optional)
3.Choose signal type for each segment and enter parameters
4.The full signal is generated and plotted
5.Apply operations interactively on the signal
## Requirements:
-MATLAB
## Output:
-Generated signal plot
-Updated signal after applying operations
